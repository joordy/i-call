## 💻 **Installation**

### Clone the repository

```bash
  git clone https://github.com/joordy/progressive-web-apps-2021.git
```

### Navigate to the repository and install the packages

```bash
  npm install
```

### Start local dev environment

```bash
 npm run dev
```

### Build export for deployment

```bash
 npm run build
```
